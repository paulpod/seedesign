Seedrs Prototype Kit

This kit lets you rapidly create HTML prototypes of interactive designs, services and layouts to see on devices.

It contains all you need to prototype anything from simple static pages to complex, data-driven transactions. Get the code from the github repository.

Pages

1. Setting out Sofia Pro on a baseline grid

2. Custom build of a Sofia Numbers font for use in tabular layouts

3. Initial layout of a complex portfolio table, scales down to 960px, suitable for use on a tablet and smaller monitors - and with visible baseline grid. On a mobile, it drops many of the columns, as well as pennies from the £numbers.

4. Some sketches for infographics exploring better ways to show the illiquid share price, with values (profit, or losses) mapped to financial events and mapping multiple (portfolio) companies values

5. Some sketches for Discovery, in the style of curated sub-indexes - see Apple iOS App store for similar

6. Quick mockup of a #hashtag or theme landing page ... beer

7. Sketches for homepage - introducing an interactive explainer, integrating 'How it works' and a calculator

About this page

We recommend you add links to your prototype to this page, rather than delete it. You'll find this page in the kit at '/app/views/index.html'.

You can change the service name by editing the file '/app/config.js'.
